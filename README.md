## 📖 Descrição do Projeto
Este projeto consiste em um sistema desktop desenvolvido em C# utilizando Windows Forms, com o objetivo de gerenciar um supermercado.

O sistema permite o cadastro de produtos, simulação de vendas através de um carrinho de compras e a emissão de uma nota simples ao final da compra.

---

## 🛠️ Tecnologias Utilizadas

- C#
- Windows Forms
- .NET Framework
- Visual Studio

---

## ✅ Funcionalidades

### 📦 Cadastro de Produtos (CRUD)
- Inserir novos produtos
- Atualizar produtos existentes
- Excluir produtos
- Listar produtos em tabela

### 🛒 Sistema de Compras
- Seleção de produtos cadastrados
- Escolha de quantidade
- Adição ao carrinho de compras
- Cálculo automático do valor total

### 🧾 Emissão de Nota
- Geração de nota com:
  - Nome do supermercado
  - Data da compra
  - Lista de produtos
  - Quantidade
  - Preço unitário
  - Total da compra
